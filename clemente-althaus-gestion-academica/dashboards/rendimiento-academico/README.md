# Dashboard de Rendimiento Académico

Análisis de logro de competencias por nivel, grado y área curricular (escala AD/A/B/C).

## Contenido esperado
- Scripts Python/pandas de análisis de notas (ej. Comité de Grado 7°–9°)
- Reportes semáforo por trimestre (primaria y secundaria)
- Comparativos vs. benchmark institucional (ej. 98% nivel B)
- Dashboards exportados (PPTX/PDF) para presentación a padres o comité

## Convención de nombres
`AAAA-Tn-nivel-grados-descripcion.ext`
Ejemplo: `2026-T2-secundaria-7-9-comite-grado.py`
