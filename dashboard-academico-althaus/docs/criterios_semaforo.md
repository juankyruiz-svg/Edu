# Criterios de semáforo de desempeño

| Color     | Umbral     | Interpretación                                      |
|-----------|------------|-----------------------------------------------------|
| Verde  🟢 | ≥ 90%      | Logro consolidado. Sostener estrategias actuales.   |
| Amarillo 🟡| 75–89%    | En proceso. Monitoreo y refuerzo preventivo.        |
| Rojo   🔴 | < 75%      | Alerta. Intervención pedagógica urgente.            |

## Indicadores

- **Índice Global de Logro** = AD + A
- **Índice de Excelencia** = AD
- **Índice de Riesgo** = B + C
